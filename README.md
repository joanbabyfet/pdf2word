## About
基于 Python3 实现批量PDF转Word功能

## Feature

* 引用 PyQt6 库, 开发桌面小工具
* 支持批量PDF转Word文件

## Requires
Python 3.11.0  
pdf2docx 0.5.6  
PyQt6 6.4.0   

## Usage
```
python main.py
```
